## Quick Lookup Patterns

### "I'm drawing a vertical stem going UP, what can follow?"
```
From: в”Ӯ (up exit)
Followers: в”Ӯ в”ғ в”¬ в”ҙ в”ң в”Ө в”ј в•· В·

Pick one:
  в”Ӯ = continue straight
  в”ң or в”Ө = branch left or right
  в•· = prepare endpoint
  В· = terminate here
```

### "I'm drawing a horizontal branch going RIGHT, what can follow?"
```
From: в”Җ (right exit)
Followers: в”Җ в”Ғ в”ң в”Ө в”ј в•ҙ В· в•ӯ в•® в•° в•Ҝ

Pick one:
  в”Җ = continue straight
  в•ҙ = prepare endpoint
  В· = terminate
  в•ӯ в•° = turn down or up
```

## Character Pick by Intent

### ENDPOINTS (Pick ONE when ready to stop drawing)
Use when branch length is done:
- `В·` вҖ“ thin leaf, small endpoint
- `вҖў` вҖ“ medium fruit/berry
- `в—Ҹ` вҖ“ large apple/fruit
- `в—Ҷ` вҖ“ star-shaped, complex fruit
- `в—Ү` вҖ“ delicate, light fruit
- `кҳҺ` вҖ“ single leaf (Unicode)

### CONTINUATIONS (What to draw while going in a direction)
- **UP or DOWN**: `в”Ӯ` (thin) or `в”ғ` (thick)
- **LEFT or RIGHT**: `в”Җ` (thin) or `в”Ғ` (thick)
- **UpRight or DownLeft**: `в•ұ` (standard) or `вҢҝ` (root-like)
- **UpLeft or DownRight**: `в•І` (standard) or `вҚҖ` (root-like)
- **WAVY VINE**: `вҲҝ` (multi-dir) or `~` (horizontal wave)

### JUNCTIONS (When splitting into multiple paths)
- **Trunk splits right**: `в”ң`
- **Trunk splits left**: `в”Ө`
- **Branch splits down**: `в”¬`
- **Branch splits up**: `в”ҙ`
- **4-way split**: `в”ј`

### CORNERS (When changing direction 90В°)
- **UPвҶ’RIGHT**: `в•°`
- **UPвҶ’LEFT**: `в•Ҝ`
- **DOWNвҶ’RIGHT**: `в•ӯ`
- **DOWNвҶ’LEFT**: `в•®`

### ROOTS (Spreading at base)
- Pattern: `в”Ӯ вҶ’ в”ң вҶ’ вҢҝ вҶ’ В·` (left)
- Pattern: `в”Ӯ вҶ’ в”Ө вҶ’ вҚҖ вҶ’ В·` (right)
- Alternative: `в”Ӯ вҶ’ в”ј вҶ’ /вҢҝ /вҚҖ \В·` (both)

### THICK STRUCTURES (Multi-cell-wide)
- `в–Ҳ` (full block) = heavy junction, knot
- `в–“` (dark shade) = softer knot, aging
- `в–Ң` `в–җ` = left/right half blocks (vertical)
- `в–Җ` `в–„` = top/bottom half blocks (horizontal)
- `в”ғ` `в”Ғ` = double-line thick variants

### ORGANIC/NATURAL (Vines, tendrils)
- `вҲҝ` (sine wave) = organic multi-dir vine
- `~` (tilde) = simple horizontal wave
- `вҢ ` `вҲ«` `вҢЎ` (integral family) = complex split clusters
- `вҢҝ` `вҚҖ` (root diagonals) = spreading organics

## Termination Rules

**Rule 1: Endpoints block all further drawing**
Once you place `В· вҖў в—Ҹ в—Ҷ в—Ү`, you cannot draw from those cells. They are final.

**Rule 2: Use stubs (в•· в•ө в•ҙ в•¶) to prepare endpoints**
Don't jump directly from `в”Ӯ` to `В·`. Go: `в”Ӯ вҶ’ в•· вҶ’ В·` (better visual flow).

**Rule 3: Combine branching with terminators**
For fruit clusters: `в”ң вҶ’ LEFT: в—Ҹ RIGHT: в—Ҹ` (fork, then add fruits on left/right).

**Rule 4: Waves can chain**
`вҲҝ вҶ’ вҲҝ вҶ’ вҲҝ вҶ’ вҖў` is valid (waves continue until a terminator).

**Rule 5: Thick blocks are junctions**
`в–Ҳ` acts like `в”ј` (all 4 directions). Use when you need a heavy knot/split.

## Visual Continuity Checklist

Before placing a character:

- [ ] **Previous char has an exit in this direction?**
  - Example: `в”Ӯ` has Up/Down exits. Can't go Left from it.

- [ ] **Next char can accept entry from opposite direction?**
  - Example: Going Right into a char вҶ’ that char must have a Left exit.

- [ ] **Is next char a terminator?**
  - If yes (`В· вҖў в—Ҹ`), then this is the final cell. No further draws after.

- [ ] **Does character family match?**
  - Don't mix thick (в–Ҳ) with thin (в”Ӯ) without a junction.
  - Don't mix wavy (вҲҝ) with straight (в”Җ) unless intentional.

- [ ] **Does depth suggest fruit size?**
  - Shallow branch вҶ’ bigger fruit (в—Ҹ)
  - Deep branch вҶ’ tiny fruit (В·)

```rust
tset_over(grid, x, y,     'в”Ӯ', color);
tset_over(grid, x, y - 1, 'в•·', color);

```

### Fork with left/right branches:
```rust

tset_over(grid, x + 2, y, 'в—Ҹ', color);  // right fruit
```

### Drooping vine with wave:
```rust

```

### Root system:
