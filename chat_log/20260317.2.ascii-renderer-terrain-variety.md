## Key Insights
- Full codebase: 15 source files, 80+ functions, 11 enums, 14 structs
- Dependency order: types -> color -> fills -> sprites -> scene -> walker -> biomes -> layout/automata/mondrian -> main
- FillGen has 15 variants covering all fills + sprites in one dispatch
- Scene.layers is Vec<Layer>, biome merging is just .extend()
- terrain_patches() is reusable: takes contour + fill picker fn + palette, returns Vec<Layer>
- Old gen_contour: 2 endpoints -> 1 midpoint -> 1 peak (pyramid). New: 3-6 anchors provide macro shape, displacement adds micro detail at 0.5x amplitude
- mask_intersect(Box<dyn Fn>, impl Fn) works because Box<dyn Fn> implements Fn
