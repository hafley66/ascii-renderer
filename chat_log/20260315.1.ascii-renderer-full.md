## What Was Taught (pre-bias for next session)
- **Truchet tiles**: 2 chars (╱╲), coin flip per cell. Emergent visual complexity from minimum information. Grid-native, zero contradiction risk.
- **DLA**: particles random-walk from spawn point, stick when adjacent to frozen cluster. Spawn NEAR cluster (not from edges) for denser growth. 3000 particles, 2000 walk steps. Three distance zones with different glyph palettes.
- **Glyph harmony**: round glyphs (○●◆) clash with angular Truchet (╱╲). Angular crystals (╳╬┼╪) blend. Braille chars (⡷⣟⢿⣻⣿) make excellent corruption zones.
- **GRIS trees**: binary recursive split. Queue-based: (x, top_y, bottom_y, depth). Splits use ╭─┼─╮. Vertical segments use │. Tips terminate at canopy_y. Bounded by max_depth and spread >> depth.
- **LLM alignment limitation**: left edges fine, right borders drift. Open-right layouts work. Small drift is tolerable. Don't fight character counting.
- **WFC vs CA vs L-systems**: WFC best for grid-native connected ornament (but needs authored adjacency rules). CA best for 1D textures/dividers. L-systems best for branching structures but angle discretization on grid is brutal.

## Design Iterations We Went Through

### /map command evolution
1. First attempt: tree-style file listing -- too boring, looked like `tree` output
2. Added box-drawing borders -- right side always drifted, removed closing borders
3. Added Japanese kanji headers 「技」-- visual distinctiveness
4. Added block characters ░▓██ -- work fine, kept them
5. Tried 2-column layout with center divider line -- replaced with flower ornaments between columns
6. Added avant garde mode: Amsterdam wrought iron (double rail ┃╻, diamond joints ◈━━●), NERV-style section headers
7. Added context gauge on left side (energy meter, AC3 style, empties from top)
8. Settled on: unique every time, shape matches topology, one screen max, no overfitting examples in the spec

### /vibes command
1. Started with prescribed examples at 3 zoom levels -- overfitting concern
2. Removed all examples, kept format rules and purpose statement
3. Added bifurcation/drift tracking -- user wants to know when conversation went off rails

### ASCII renderer POC iterations
1. Truchet only -- works but boring, just noise
2. Added DLA from edges -- too sparse, particles waste time in empty space
3. Switched to spawning near cluster -- much denser crystal growth
4. Round crystal glyphs (○●◆◈◇) -- clashed with angular Truchet
5. Angular glyphs (╳╬┼╪▪) -- blended much better
6. Three-zone system: border (structural ╍┅┉), mid (geometric ⟋⟍⧸⧄⧈), far (braille corruption ⡷⣟⢿⣻) -- visual depth
7. Added GRIS-style trees -- box-drawing binary splits, flat canopy
8. Added flower stamps -- 5 styles (diamond, circle, star, box, braille)
9. Full combined mode: Truchet bg + trees in corners + content box + flowers
