## Key Insights
- TerrainContext pattern: Scene + domain-specific data (contours) for post-pass. Other biomes will follow same pattern where needed.
- Scene.layers is Vec<Layer>, so merging biome scenes is just .extend() -- world mode becomes: generate N biome scenes, merge layer stacks with spatial masks, render once
- terrain_patches() is generic enough to reuse for any contour band: takes contour, fill picker fn, palette, returns Vec<Layer>
- Multi-anchor contour gen: seed 3-6 points, interpolate, then run displacement at reduced amplitude. Anchors provide macro shape, displacement adds micro detail.
- mask_intersect(Box<dyn Fn>, impl Fn) works because Box<dyn Fn> implements Fn
