## Key Insights
- connect_glyph was the root cause of all pen tree disconnection. Every turn char was wrong. The table confused "arriving from direction" with "traveling in direction".
- Trees don't care about overwrite rules -- they just draw. Layering is the compositor's problem. Removed can_overwrite from trait.
- should_branch needs integer index+count, not float fractions. ASCII grid = integer coordinates = integer intervals.
- Randomness from outside: tree is pure function of (TreeParams + rng state). Tree consumes entropy, doesn't create it.
- Spiral tree uses default growth loop. Split and candelabra override grow() entirely.
- Flowers should NOT use TreePen (box-drawing chars look like branches). Use radial direct placement instead.
- Stem and bloom grow on different energy curves (stem linear, bloom accelerates 0.4-0.7).
