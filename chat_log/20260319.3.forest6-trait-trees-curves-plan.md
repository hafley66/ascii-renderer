## Key Insights
- Haiku agents in worktree isolation write code to the actual working directory (not ephemeral). Need to review their output for: wrong rng usage, missing params.spread() caps, state communication between trait methods.
- StormTree needed StdCell<i32> to carry lean direction from draw_trunk to should_branch/draw_branch -- this is a code smell the TrunkAlgo extraction will fix.
- Single-cell diagonals (╱╲) create visual gaps in trunks. 2-column curve pairs (╰╭ / ╯╮) with manual pen.x adjustment maintain continuity.
- The flower sprite rotated horizontal could be a trunk pattern -- trunks are just organic line algorithms, same as branches.
- Forest4 composition quality comes from: anchor tree, height tiers (scrubby/medium/towering), spread tiers, min spacing, depth brightness, leaf clusters, flower scatter, foreground layer.
