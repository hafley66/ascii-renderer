## Goal
Bifurcate bole styles into compact tree boles (2-3 rows) and full-size standalone bush sprites. Port remaining forest4 personality trees to TreeDrawer system. Add trees4 and bushes modes.

## Key Insights
- Boles were originally designed as root flares but grew to 5-25 rows. The full-size versions are genuinely interesting as standalone bush/shrub sprites rather than being deleted.
- Post-pass gradient (draw flat then sweep bounding box with lerp) is non-invasive and works well as scaffold. Eventually color should be driven per-glyph by the energy algorithm.
- User specifically wants ChevronTree -- the layered V-shape algorithm is too good to live only as a compacted root flare.
- SplitTree's identity is about splitting, not about trunk length. User is viscerally annoyed by long trunks on it.
- Squat bole styles (18-23) were designed as minimal 1-row flares and don't work well as standalone bushes. Could filter from bushes mode.

## Open Questions
- How short should SplitTree trunk be? User wants splitting "sooner" even after 1/8 + 2-cell stem
- Should bushes mode filter out squat styles 18-23 or label them differently?
- Energy-driven color gradient design: how should energy map to hue/saturation/lightness per glyph position?
