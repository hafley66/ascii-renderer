## Next: TileEdgeBehavior Trait
Design agreed. Each tile variant gets bespoke edge control via trait with full algorithm context.

### The context struct the trait receives:
```rust
struct TileEdgeContext {
    gx: usize, gy: usize,           // absolute grid position
    rx: i32, ry: i32,               // position relative to rect origin (can be negative)
    tx: usize, ty: usize,           // tile-local coords (phase-shifted, stagger-applied), 0..period
    tile_col: i32, tile_row: i32,   // which repetition (can be negative outside rect)
    stagger_group: usize,           // current stagger group index
    stagger_offset: usize,          // current stagger x-offset in cells
    normal_char: char,              // what normal sampling returns at this coord
    normal_ci: u8,                  // color index normal sampling returns
    dist_left: i32,                 // distance from left edge (negative = inside by N)
    dist_right: i32,                // distance from right edge
    dist_top: i32,                  // distance from top edge
    dist_bottom: i32,               // distance from bottom edge
    skew: u32,                      // 0-100 skew intensity
    extend: usize,                  // max bleed distance in cells
}
```

### Trait interface:
```rust
trait TileEdgeBehavior {
    fn at_edge(&self, ctx: &TileEdgeContext, rng: &mut StdRng) -> Option<(char, u8)>;
}
```

### Per-variant planned behaviors:
- Seigaiha: close arches by mirroring ty at bottom edge
- Shippo: close circles by mirroring
- Nowaki: extend individual strokes (│ and ╱) past edge with length decay
- GrannySquare: close boxes with └┘ pairs at bottom
- CrocodileScale: close scales by mirroring V shape
- BishamonKikko: close hexagons at bottom
- ShellStitch: close arches like seigaiha
- Asanoha/Yabane/Higaki: generic dropout (self-similar patterns, no closure needed)

### Dispatch in fill_tile_ex:
```rust
if dist > 0.0 || near_edge {
    match variant.at_edge(&ctx, rng) {
        Some((ch, ci)) => draw it,
        None => continue (skip/dropout),
    }
} else {
    normal tile sampling
}
```

Default impl = current generic dropout. Overrides are opt-in per variant.
skew controls intensity: 0 = no edge method called, continuous 0-100 or discrete thresholds.
