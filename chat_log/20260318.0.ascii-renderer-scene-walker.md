## Key Insights
- `use<>` in Rust 2024: `impl Fn() -> f32 + use<>` means "captures no lifetimes from inputs." Needed when closure copies fields from a reference but the return type would otherwise inherit the borrow lifetime.
- fill_masked was a compositing bug -- it blanked cells outside mask, destroying previous layers. Save/restore fix makes layers truly compositable.
- User wants: distinct island worlds along a path, trees/faces/diamonds breaking out of bounds, tiling/noise staying bounded inside. Reference: radial density diffusion, not BSP grid fill.
- fill_breaks_out() classifies FillGen variants: Tree/Flower/Fruit/AztecDiamond/Fret/Mask break out; Tile/Noise/Crosshatch/etc stay clipped.
- Growth patterns (Rect/Diamond/FlatHex) control element placement within each box -- spiral offsets from center.
- Overlap dedup: overlapping boxes cannot reuse same (fill_discriminant, color_index) combo.
